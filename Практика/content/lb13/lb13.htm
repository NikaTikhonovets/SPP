Лабораторная работа №8
Задание:

Подключить к интерфейсу фреймворк AngularJS. Добиться того, чтобы работа с каждой новой формой была возможна без перезагрузки страницы (одностраничное приложение).


Данные с сервера должны приходить в виде JSON-объектов. При помощи встроенных в AngularJS механизмов binding-а обеспечить корректную работу с формами.


Добавить анимацию элементов интерфейса с использованием библиотеки jQuery.


Контрольные вопросы:

  1. Назначение библиотеки jQuery.
  2. Особенности фреймворка AngularJS.
