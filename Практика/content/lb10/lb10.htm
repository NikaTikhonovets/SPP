Лабораторная работа №5
Задание:

Используя современные возможности HTML5 и CSS3, оформить интерфейс приложения в едином стиле. Каждая страница должна содержать:

  • header с информацией о приложении,
  • navigation с быстрыми переходами между страницами приложения,
  • footer с информацией о разработчиках,
  • content, отображающий интерфейс взаимодействия с объектами сущностей приложения.


Верстка должны быть кроссбраузерной и отзывчивой. Для тестирования верстки использовать браузеры Google Chrome, Mozilla Firefox, Internet Explorer 9+.


Для обеспечения кроссбраузерности и отзывчивости можно использовать фреймворк Bootstrap либо подобные ему, использующие многоколоночную верстку.


В интерфейсе должно отображаться имя пользователя, вошедшего в систему.
Удаление любых сущностей должно подтверждаться при помощи модального окна.


Приветствуется использование метаязыков SASS (SCSS) или LESS для написания собственных стилей.


Контрольные вопросы:

  1. Способы создания отзывчивой верстки.
  2. Особенности верстки для различных браузеров, включая Internet Explorer.
